Renesas Envision Kit RX65N ガジェット
=========
![R5F564ML](RayTracerRTK5.jpg)

## 概要
Envision Kit RX65N を使ったレイ・トレーサー
   
## プロジェクト・リスト
 - main.cpp
 - Makefile
 - raytracer.hpp
      
## ビルド方法
 - make する。
 - raytracer.mot ファイルを書き込む。   

## 操作と動作
 - SW2 を押すと、画面をクリアして、再度レンダリングを行う。
 - レンダリングを行う度に、フルスクリーン（480x272）と（320x240）を切り替える。
