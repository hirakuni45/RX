
## RX23T features / 特徴

- RXv2 core / RXv2 コア
- Maximum operating frequency 40MHz / 最大動作周波数 40MHz
- Single-precision 32-bit floating point / 32 ビット単精度浮動小数点
- 2.7V ～ 5.5V Operation / 動作
- 12 Bits A/D / １２ビットＡ／Ｄ変換器
- 8 Bits D/A (for Compalator) / ８ビットＤ／Ａ変換器（コンパレーター用）
